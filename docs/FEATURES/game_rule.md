# 게임 룰 상세

## 클리어 조건

화면 내 모든 풍선을 터뜨리면 스테이지 클리어.

## 사망 조건

| 조건 | 내용 |
|------|------|
| 풍선 접촉 | 크기에 관계없이 풍선에 닿으면 즉시 사망 |
| 시간 초과 | 제한 시간(타이머)이 0이 되면 사망 |

사망 시 해당 스테이지 처음부터 재시작. 잔기를 모두 소모하면 게임 오버.

---

## 풍선 (Balloon)

### 크기 및 분열 규칙

| 크기 | 바운드 높이 | 격파 시 |
|------|------------|---------|
| 4 (대형) | 최대 | 크기 3 × 2개로 분열 |
| 3 (중형) | 중간 | 크기 2 × 2개로 분열 |
| 2 (소형) | 낮음 | 크기 1 × 2개로 분열 |
| 1 (최소형) | 바닥 근처 | 완전 소멸 |

- 풍선은 좌우로 이동하면서 바닥/벽에 부딪히면 반사
- 크기가 클수록 바운드 호(arc)가 높고 느림
- 크기가 작을수록 낮고 빠르게 움직임

### 이동 방향
- 스테이지 시작 시 각 풍선에 무작위 좌/우 방향 부여
- 벽 충돌 시 수평 방향 반전, 바닥 충돌 시 위로 튀어오름

---

## 플레이어

- 좌우 이동만 가능, 점프 없음
- 풍선이나 적에 닿으면 즉시 사망
- 발사 키를 누르면 위 방향으로 와이어 발사

### 조작 키 (게임 중)

| 키 | 동작 |
|----|------|
| ← / → | 좌우 이동 |
| Space / Z | 와이어 발사 |

---

## 무기: Harpoon Wire (기본)

- 발사하면 와이어가 달린 작살이 수직 위로 올라감
- 천장에 닿으면 와이어가 사라짐
- **풍선에 직접 맞거나** 늘어난 **와이어에 풍선이 닿아도** 격파
- 이전 와이어가 화면에 남아 있는 동안 재발사 불가 (1발 제한)

---

## 점수 시스템

### 풍선 크기별 기본 점수

작은 크기일수록 점수가 높다.

| 크기 | 점수 |
|------|------|
| 4 (대형) | 낮음 |
| 3 (중형) | 보통 |
| 2 (소형) | 높음 |
| 1 (최소형) | 가장 높음 |

### 연속 격파 배율

같은 발사로 연속 격파 시 배율 적용:

```
1번째: x1 → 2번째: x2 → 3번째: x4 → 4번째 이상: x8
```

## 잔기

- 기본 잔기 3개로 시작
- 잔기를 모두 소모하면 게임 오버 → 메인 화면으로 복귀
