# 미션 1: 관악산 (Mt. Gwanak)

## 스테이지 기본 정보

| 항목 | 내용 |
|------|------|
| 배경 | 한국 관악산 (낮) |
| 난이도 | ★☆☆☆☆ (입문) |
| 제한 시간 | 60초 |
| 초기 잔기 | 3개 |

---

## 초기 풍선 구성

| 풍선 크기 | 수량 |
|----------|------|
| 4 (대형) | 2개  |

대형 풍선 2개가 화면 좌우에서 시작. 각각 격파 시 분열을 반복하여 최대 16개(최소형 기준)까지 증가한다.

### 분열 트리

```
대형(4) ──┬──▶ 중형(3) ──┬──▶ 소형(2) ──┬──▶ 최소형(1) 소멸
          │              │              └──▶ 최소형(1) 소멸
          │              └──▶ 소형(2) ──┬──▶ 최소형(1) 소멸
          │                             └──▶ 최소형(1) 소멸
          └──▶ 중형(3) ──┬──▶ 소형(2) ──┬──▶ 최소형(1) 소멸
                         │              └──▶ 최소형(1) 소멸
                         └──▶ 소형(2) ──┬──▶ 최소형(1) 소멸
                                        └──▶ 최소형(1) 소멸
```

대형 1개 기준 최소형 8개로 분열 → 총 2개 기준 최대 동시 풍선 수: **16개**

---

## 지형

- 평탄한 단일 바닥 플랫폼
- 중간 발판 없음, 장애물 없음
- 벽(좌/우)과 천장(상)은 풍선이 반사되는 경계

---

## 난이도 특성

- 풍선 이동 속도: 느림
- 풍선 바운드 패턴: 단순 (무작위 방향 반전 없음)
- 특수 풍선 없음
- 적(몬스터) 없음
- 아이템 드롭: 없음 (기본 와이어만 사용)

---

## 클리어 흐름

```
시작
 └─ 대형 풍선 2개 등장 (좌/우 방향으로 이동)
      └─ 와이어로 격파 → 분열
           └─ 분열된 풍선 반복 격파
                └─ 화면 내 풍선 전부 소멸
                     └─ 스테이지 클리어
```

## 게임 오버 조건

- 풍선에 접촉 → 잔기 1 감소 → 잔기 0이 되면 게임 오버
- 60초 이내 클리어 실패 → 사망 처리
