# 메인 화면 구성

## 화면 레이아웃

```
┌─────────────────────────────────┐
│                                 │
│         [ PANG 로고 ]           │
│                                 │
│       HI-SCORE : 000000         │
│                                 │
│         [ 1 PLAYER ]            │
│         [ 2 PLAYER ]            │
│                                 │
│   (풍선 애니메이션 배경)         │
│                                 │
│    Press any key to start       │
│                                 │
└─────────────────────────────────┘
```

## 구성 요소

### 타이틀 로고
- 화면 상단 중앙에 "PANG" 타이틀 표시

### 최고 점수 (HI-SCORE)
- 로고 아래에 현재 세션의 최고 점수 표시
- 초기값: 000000

### 게임 시작 메뉴
- **1 PLAYER**: 1인 플레이 시작
- **2 PLAYER**: 2인 동시 플레이 시작 (미션 1 범위에서는 1P만 구현)
- 키보드 방향키로 항목 선택, Enter 또는 Space로 확정

### 배경
- 풍선이 튀어 다니는 애니메이션으로 게임 분위기 연출

## 전환 흐름

```
메인 화면 → (1 PLAYER 선택) → 미션 1 시작
```

## 조작 키 (메인 화면)

| 키 | 동작 |
|----|------|
| ↑ / ↓ | 메뉴 항목 선택 |
| Enter / Space | 선택 확정 |
