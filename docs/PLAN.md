    # PLAN: PANG 게임 개발 Phase별 목표

각 Phase는 **브라우저에서 직접 실행하고 테스트할 수 있는 동작하는 SW**를 목표로 합니다.

---

## Phase 1: 메인 화면

**목표:** 게임을 처음 실행하면 메인 화면이 나온다.

- PANG 타이틀 로고 표시
- HI-SCORE 표시 (초기값 000000)
- 메뉴 항목: **1 PLAYER** / **게임 종료**
- ↑ ↓ 키로 메뉴 이동, Enter로 선택
- 마우스 오버 시 항목 선택, 클릭 시 즉시 실행
- 게임 종료 선택 시 브라우저 탭 종료 안내 문구 표시

**고객님 확인 포인트**
> 메인 화면이 뜨는지, 메뉴 항목 간 이동이 잘 되는지 확인해 주세요.
> 선택 항목이 시각적으로 구분되는지(색상 또는 커서 표시) 봐주세요.
> 마우스로 항목에 올리면 선택되고, 클릭하면 실행되는지 확인해 주세요.

---

## Phase 2: 게임 캔버스 및 배경

**목표:** 1 PLAYER를 선택하면 게임 화면으로 전환된다.

- 메인 화면 → 게임 화면 전환
- 관악산 배경 이미지 렌더링
- 60fps 게임 루프 동작
- 바닥/벽/천장 경계선 표시

**고객님 확인 포인트**
> 1 PLAYER를 선택했을 때 게임 화면으로 전환되는지 확인해 주세요.
> 배경이 자연스럽게 보이는지, 화면 비율이 어색하지 않은지 봐주세요.

---

## Phase 3: 플레이어 등장 및 이동

**목표:** 캐릭터가 나타나고 키보드로 움직인다.

- 플레이어 캐릭터 화면 하단 중앙에 렌더링
- `←` `→` 키로 좌우 이동
- 좌우 벽에서 이동 멈춤

**고객님 확인 포인트**
> 캐릭터가 키 입력에 즉시 반응하는지 확인해 주세요.
> 이동 속도가 너무 빠르거나 느리다고 느껴지면 피드백 주세요.

---

## Phase 4: 풍선 등장 및 물리

**목표:** 풍선이 화면 안에서 튀어다닌다.

- 대형(4단계) 풍선 2개 화면에 등장
- 좌우 이동 + 바닥·벽 반사 물리
- 풍선이 위아래로 호(arc)를 그리며 튀는 동작

**고객님 확인 포인트**
> 풍선이 벽과 바닥에서 자연스럽게 튕기는지 확인해 주세요.
> 풍선이 화면 밖으로 나가거나 겹치는 이상 동작이 없는지 봐주세요.

---

## Phase 5: 와이어 발사

**목표:** 무기를 위로 쏠 수 있다.

- `Space` 키로 와이어 발사
- 와이어가 수직 위로 올라가 천장에 닿으면 사라짐
- 이전 와이어가 사라지기 전까지 재발사 불가 (1발 제한)

**고객님 확인 포인트**
> Space를 눌렀을 때 와이어가 위로 올라가는지 확인해 주세요.
> 와이어가 천장에 닿으면 사라지는지, 연속으로 빠르게 눌렀을 때 1발만 유지되는지 봐주세요.

---

## Phase 6: 풍선 격파 및 분열

**목표:** 와이어로 풍선을 터뜨리면 분열된다.

- 와이어(또는 와이어 줄)와 풍선 충돌 감지
- 격파 시 한 단계 작은 풍선 2개로 분열
- 최소형(1단계) 격파 시 완전 소멸
- 화면 내 모든 풍선 소멸 시 **스테이지 클리어** 문구 표시

**고객님 확인 포인트**
> 풍선을 쐈을 때 분열이 되는지, 최소형이 소멸되는지 확인해 주세요.
> 분열된 풍선들이 양옆으로 자연스럽게 퍼지는지 봐주세요.
> 모든 풍선을 없애면 클리어 문구가 뜨는지 확인해 주세요.

---

## Phase 7: 플레이어 사망 및 잔기

**목표:** 풍선에 닿으면 죽고, 잔기가 줄어든다.

- 풍선-플레이어 충돌 감지 → 사망 처리
- 잔기 3개, 화면 상단에 잔기 아이콘 표시
- 사망 후 리스폰 (잔기 소모)
- 잔기 0 → **게임 오버** 화면 표시 후 메인 화면 복귀

**고객님 확인 포인트**
> 풍선에 닿았을 때 즉시 사망이 되는지 확인해 주세요.
> 잔기가 화면에서 줄어드는 것이 보이는지, 잔기 0이 됐을 때 게임 오버 화면이 뜨는지 봐주세요.
> 게임 오버 후 메인 화면으로 돌아오는지 확인해 주세요.

---

## Phase 8: 제한 시간

**목표:** 제한 시간 안에 클리어하지 못하면 죽는다.

- 화면 상단에 60초 카운트다운 표시
- 시간 초과 시 사망 처리 (잔기 소모)

**고객님 확인 포인트**
> 타이머가 줄어드는지, 0초가 됐을 때 사망 처리가 되는지 확인해 주세요.
> 사망 후 타이머가 초기화되어 다시 시작되는지 봐주세요.

---

## Phase 9: 점수 시스템

**목표:** 풍선을 터뜨리면 점수가 오른다.

- 풍선 크기별 점수 부여 (작을수록 고점수)
- 연속 격파 배율 적용 (x1 → x2 → x4 → x8)
- 화면 상단에 현재 점수 표시

**고객님 확인 포인트**
> 풍선을 터뜨릴 때마다 점수가 오르는지 확인해 주세요.
> 작은 풍선이 더 높은 점수를 주는지, 연속으로 터뜨렸을 때 점수가 더 빠르게 오르는지 봐주세요.

---

## Phase 10: 미션 1 완성 및 전체 흐름 연결

**목표:** 처음부터 끝까지 한 판을 완주할 수 있다.

- 클리어 화면 → 메인 화면 복귀
- HI-SCORE 갱신 (세션 내 최고 점수 유지)
- 메인 → 게임 → 클리어/게임오버 → 메인 전체 흐름 연결

**고객님 확인 포인트**
> 메인 화면에서 시작하여 미션 1을 클리어하거나 게임 오버까지 전체를 한 번 플레이해 주세요.
> 화면 전환이 어색하거나 끊기는 부분이 없는지, HI-SCORE가 갱신되는지 전반적으로 봐주시면 됩니다.
