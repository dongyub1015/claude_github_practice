# PRD: PANG 게임 - 미션 1 구현

## 게임 개요

**PANG** (팡, 일본명: Pomping World)은 1989년 Mitchell Corporation이 개발한 아케이드 액션 게임이다.
플레이어는 화면 안에서 튀어 다니는 풍선(버블)을 모두 터뜨려 스테이지를 클리어하는 것이 목표다.

---

## 핵심 게임 규칙

- 화면 내 모든 풍선을 터뜨리면 스테이지 클리어
- 풍선에 닿으면 즉시 사망
- 제한 시간 내에 클리어하지 못하면 사망
- 잔기를 모두 소모하면 게임 오버

---

## 게임 오브젝트

### 풍선 (Balloon)

풍선은 4단계 크기로 구분되며, 터뜨리면 한 단계 작은 풍선 2개로 분열된다. 최소 크기(1단계)는 터뜨리면 완전히 소멸한다.

| 크기 | 특징 |
|------|------|
| 4 (대형) | 가장 높이 튀어오름 |
| 3 (중형) | 중간 높이로 튐 |
| 2 (소형) | 낮게 튐 |
| 1 (최소형) | 바닥 근처에서 튐, 소멸 시 가장 높은 점수 |

### 플레이어

- 좌우 이동만 가능 (점프 불가)
- 위 방향으로만 무기 발사

### 기본 무기 (Harpoon Wire)

- 수직 위로 와이어가 달린 작살 발사
- 풍선에 직접 맞거나 와이어에 닿아도 격파 가능
- 이전 와이어가 화면에 있는 동안 재발사 불가

---

## 미션 1 스펙

| 항목 | 내용 |
|------|------|
| 배경 | 일본 후지산 (낮) |
| 초기 풍선 | 대형(4단계) 풍선 2개 |
| 지형 | 평탄한 단일 플랫폼, 장애물 없음 |
| 난이도 | 입문 (풍선 속도 느림) |

대형 풍선 1개는 최종적으로 최소형 풍선 8개(2→4→8)까지 분열 가능하므로, 미션 1에서 처리해야 할 최대 풍선 수는 16개다.

---

## 점수 시스템

- 풍선 크기가 작을수록 격파 점수가 높다
- 연속 격파 시 배율 증가: x1 → x2 → x4 → x8

---

## 미션 1 클리어 흐름

```
게임 시작
  └─ 대형 풍선 2개 등장
       └─ 풍선 격파 → 분열 반복
            └─ 화면 내 풍선 전부 소멸 → 스테이지 클리어
```
