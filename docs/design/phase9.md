# 설계: Phase 9 — 점수 시스템

## 목표 요약

풍선 격파 시 크기별 기본 점수에 연속 격파 배율을 곱한 점수를 부여하고, HUD 중앙에 현재 점수를 표시한다.

---

## 풍선 크기별 기본 점수

| 크기 | 점수 |
|------|------|
| 4 (대형) | 100 |
| 3 (중형) | 200 |
| 2 (소형) | 400 |
| 1 (최소형) | 800 |

---

## 연속 격파 배율 (Combo)

와이어 발사마다 성공/실패 여부로 콤보를 관리한다.

| 누적 성공 횟수 | 배율 |
|--------------|------|
| 1번째 | x1 |
| 2번째 | x2 |
| 3번째 | x4 |
| 4번째 이상 | x8 |

```
배율 = min(2^(combo - 1), 8)
획득 점수 = BALLOON_SCORES[size] × 배율
```

### 콤보 리셋 조건

| 조건 | 처리 |
|------|------|
| 와이어가 천장에 닿아 소멸 (미격파) | combo = 0 |
| 플레이어 사망 | combo = 0 |

---

## 상태 추가

```ts
const scoreRef = useRef(0)   // 현재 점수
const comboRef = useRef(0)   // 누적 연속 격파 수
```

---

## 점수 계산 위치 (게임 루프)

와이어-풍선 충돌 처리 블록에서 격파 확정 시 점수 부여:

```
hit 확정 시:
  comboRef.current += 1
  multiplier = min(2^(combo-1), 8)
  scoreRef.current += BALLOON_SCORES[b.size] * multiplier
```

와이어가 천장 소멸 후 (미격파):
```
wasActive && !wire.active && !hitThisFrame → comboRef = 0
```
